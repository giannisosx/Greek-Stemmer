# Greek-Stemmer

A stemmer for the greek language written in Python, based on scientific publications by <a href="http://sais.se/mthprize/2007/ntais2007.pdf">George Ntais</a>, <a href="http://uta32-kk.lib.helsinki.fi/bitstream/handle/10024/80480/gradu03463.pdf?sequence=1">Spyridon Saroukos</a> and a similar script by the  <a href="https://github.com/skroutz/greek_stemmer">Skroutz Team</a>.
